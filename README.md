# 🛍️ Sistema de Punto de Venta - Proyecto Didáctico Python

## 📋 Descripción General

¡Bienvenido/a al **Sistema de Gestión de Stock para Tienda de Ropa**! 👗👕✨

Este proyecto es una **herramienta educativa completa** diseñada para aprender **Python** y el desarrollo de **interfaces gráficas con Tkinter** desde cero. El sistema simula un punto de venta básico para una tienda de ropa, permitiendo gestionar productos, calcular precios y administrar el inventario.

## 🎯 Objetivos de Aprendizaje

### 🐍 **Fundamentos de Python**
- ✅ Manipulación de listas y estructuras de datos
- ✅ Funciones y lógica de programación
- ✅ Manejo de excepciones y validaciones
- ✅ Operaciones matemáticas y cálculos comerciales

### 🖥️ **Interfaces Gráficas con Tkinter**
- ✅ Creación de ventanas y frames
- ✅ Implementación de widgets (labels, entries, buttons, checkbuttons)
- ✅ Gestión de eventos y comandos
- ✅ Navegación entre diferentes pantallas
- ✅ Cuadros de diálogo y mensajes emergentes

## 🚀 Funcionalidades Principales

### 🔐 **Sistema de Autenticación**
- Login seguro con usuario y contraseña
- Validación de credenciales
- Transición entre interfaces

### 📊 **Gestión de Inventario**
- Visualización completa del stock disponible
- Información detallada por producto (nombre, precio, cantidad, disponibilidad)
- Cálculo automático del valor total por producto

### 💰 **Cálculo de Precios**
- Sistema de venta con múltiples productos
- Cálculo de subtotales y total general
- Manejo de cantidades variables

### ➕ **Administración de Productos**
- Formulario para agregar nuevos productos
- Campos para nombre, precio, cantidad y disponibilidad
- Actualización en tiempo real del inventario

## 🛠️ Conceptos Técnicos Implementados

### **Estructuras de Datos**
- Listas para almacenamiento de información de productos
- Variables booleanas para estados de disponibilidad
- Entradas de usuario mediante widgets Tkinter

### **Lógica de Programación**
- Control de flujo con condicionales
- Iteraciones sobre estructuras de datos
- Funciones modularizadas para cada operación
- Manejo de errores con try-except

### **Interfaz de Usuario**
- Diseño responsive con frames
- Organización jerárquica de elementos
- Mensajes de confirmación y error
- Navegación intuitiva entre secciones

## 📁 Estructura del Proyecto

```
sistema_punto_venta/
```

3. **Validar entradas** de usuario de forma segura
